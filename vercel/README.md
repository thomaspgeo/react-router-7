## Getting Started

### Installation

Choose your preferred package manager and run one of the following commands:

```bash
# Using npm
npm install

# Using pnpm
pnpm install

# Using Bun
bun install
```

### Development

Start the development server with HMR:

```bash
# Using npm
npm run dev

# Using pnpm
pnpm dev

# Using Bun
bun dev
```

Your application will be available at `http://localhost:5173`.

## Building for Production

Create a production build:

```bash
# Using npm
npm run build

# Using pnpm
pnpm build

# Using Bun
bun run build
```
